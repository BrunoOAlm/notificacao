# 🔔 Microsserviço de Notificação

Este projeto faz parte de um sistema distribuído de Agendador de Tarefas baseado em arquitetura de microsserviços.

Ele é responsável pelo envio e gerenciamento de notificações dentro do sistema.

---

## 📌 Sobre o projeto

Este microsserviço tem como objetivo lidar com notificações do sistema, podendo futuramente enviar alertas sobre tarefas, eventos ou atualizações importantes para o usuário.

---

## 📌 Responsabilidades

- Gerenciamento de notificações
- Registro de eventos de notificação
- Processamento de mensagens recebidas de outros serviços
- Exposição de API para consulta de notificações

---

## 📌 Arquitetura do sistema

Frontend → BFF → Microsserviços  
Este serviço integra o ecossistema do Agendador de Tarefas.

---

## 📌 Tecnologias utilizadas
